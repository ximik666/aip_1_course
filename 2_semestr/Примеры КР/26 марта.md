## Задание 1. Обработка списка (15 баллов)

### Условие
Дан список целых чисел. Необходимо:
1. Удалить из списка все отрицательные числа
2. Найти среднее арифметическое оставшихся элементов
3. Вывести:
   - новый список
   - среднее арифметическое

### Входные данные
Список задаётся в программе:
[3, -1, 5, -7, 8, 0]

### Выходные данные
Список без отрицательных чисел и среднее значение

### Пример вывода
[3, 5, 8, 0]
Среднее: 4.0

## Задание 2. Сортировка выбором (15 баллов)

### Условие
Реализовать алгоритм сортировки выбором (selection sort) самостоятельно.

Нельзя использовать:
- sorted()
- list.sort()

### Входные данные
Список:
[7, 2, 9, 1, 5]

### Что нужно сделать
Отсортировать список по возрастанию

### Выходные данные
Отсортированный список

### Пример вывода
[1, 2, 5, 7, 9]

## Задание 3. Работа с файлом (20 баллов)

### Условие
В файле numbers.txt находятся целые числа (по одному в строке).

Необходимо:
1. Считать все числа из файла в список
2. Найти минимальное и максимальное число
3. Записать результат в файл result.txt

### Формат входного файла (numbers.txt)
3
7
1
9

### Формат выходного файла (result.txt)
max: 9
min: 1

## Задание 4. Стек (25 баллов)

### Условие
Реализовать стек с использованием списка.

Стек должен поддерживать операции:
- добавление элемента (push)
- удаление элемента (pop)

### Что нужно сделать
1. Создать пустой стек
2. Добавить элементы: 1, 2, 3, 4
3. Удалить два элемента
4. Вывести текущее содержимое стека

### Выходные данные
Содержимое стека (снизу вверх)

### Пример вывода
[1, 2]

---

## Задание 5. Очередь (25 баллов)

### Условие
Реализовать очередь с использованием списка.

Очередь должна поддерживать:
- добавление элемента в конец
- удаление элемента из начала

### Что нужно сделать
1. Создать пустую очередь
2. Добавить элементы: 10, 20, 30
3. Удалить один элемент
4. Вывести текущую очередь

### Выходные данные
Список элементов очереди

### Пример вывода
[20, 30]
